An advanced Burtle guide

This will be a very simple and small guide for using Burtle.


Step 1: Import burtle

This time we will be using BScreen. Image BScreen as your entire window, its a powerful object that we will learn more about later.


Step 2: Create multiple burtles

Your own images go into the brackets after "Burtle", white_circle is a default preset though.

Give your burtle a friendly name such as "ball", it will reduce confusion later on.


Step 3: Fix the paddle

By default a paddle will be horizontal, which means that we will have to flip it upright by rotating it 90 degrees.

As the "left_paddle" is supposed to be on the left, we will move it to the left 150 pixels. Remember that the ball is still in the middle, as we have not moved it yet.


Step 4: Make the ball touch the paddle

Here we want the ball to keep moving to the left, so we will use the ".go()" method, to make ball go to the left. The difference here is that it is running in an infinite while loop, therefore making it constantly move to the left.


Step 5: Collision detection

The "is_hitting" method will return either True or False depending if the burtle is hitting the burtle given as a parameter.
