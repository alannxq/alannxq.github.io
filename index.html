Current Methods

l angle → Look left

r angle → Look right

tp burtle → Teleport to another burtle

look direction → Sets the heading of your turtle

draw_grid list → Given a 2d list, it will draw a grid

change_image image_path → Changes the image of your burtle

convert_to_gif image_path → Converts the given image into a gif format

go direction distance → Move the burtle in a direction of your chosing

change_size int → 50, halfs the size of your burtle image, -100 doubles it

draw_square x y size color → Draws a square, all arguments are optional

is_hitting burtle → Returns true or false depending if its hitting the given burtle

is_clicked x y → Return true or false depending if the given x and y are with the burtle's hitbox
